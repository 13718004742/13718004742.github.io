OC-算法之插入排序
发表于2020-08-07|更新于2020-08-09
|字数总计:2.9k|阅读时长:10分钟|阅读量:

基本思想

每步将一个待排序的对象，按其关键码大小，插入到前面已经排好序的一组对象的适当位置上，知道全部对象都插入为止。即边插入边排序，保证子序列中随时都是排好序的。

基本操作

有序插入

  • 在有序列序列中插入一个元素，保持序列有序，有序长度不断增加。
  • 可以插入在中间、最前面和最后面

插入排序种类

1.顺序法定位插入位置 – 直接插入排序

2.二分法定位插入位置 – 二分法插入排序

3.缩小增量多遍插入排序 – 希尔排序

直接插入排序

1.有一个已经有序的数据序列，要求在这个已经排好的数据序列中插入一个数，但要求插入后此数据序列仍然有序，这个时候就要用到一种新的排序方法——插入排序法,插入排序的基本操作就是将一个数据插入到已经排好序的有序数据中，从而得到一个新的、个数加一的有序数据，算法适用于少量数据的排序，时间复杂度为O(n^2)。是稳定的排序方法。插入算法把要排序的数组分成两部分：第一部分包含了这个数组的所有元素，但将最后一个元素除外（让数组多一个空间才有插入的位置），而第二部分就只包含这一个元素（即待插入元素）。在第一部分排序完成后，再将这个最后元素插入到已排好序的第一部分中。

插入排序的基本思想是：每步将一个待排序的记录，按其关键码值的大小插入前面已经排序的文件中适当位置上，直到全部插入完为止。

总结:插入排序法（Inser Sort）是将一个数据插入到已经排好序的有序数据中，从而得到一个新的、个数加一的有序数据，算法适用于少量数据的排序。

2.算法思想

  • 从第一个元素开始，该元素可以认为已经被排序
  • 取出下一个元素，在已经排序的元素序列中从后向前扫描
  • 如果该元素（已排序）大于新元素，将该元素移到下一位置
  • 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置
  • 将新元素插入到下一位置中
  • 重复步骤2~5

4.算法分析

  • 直接插入排序的算法性能

  • 时间复杂度

1.最好的情况（关键字在记录中顺序有序）：
1)当元素的初始序列为正序时，仅外循环要进行n-1趟排序且每一趟只进行一次比较，没有进入if语句不存在元素之间的交换(移动)。此时比较次数(Cmin)和移动次数(Mmin)达到最小值。 此时时间复杂度为O(n)。

2)举例：
如：11 20 32 前三个有序 47 50 后两个有序
i-1 i
比如说第i个元素47 和 第i-1个元素32进行比较1次， 由于比i-1个元素大，不需要移动所以移动的次数为 0

再比如下一个元素50 和 47元素比较1次，由于50比47大，不需要移动所以移动的次数为0

以此类推，第一个不需要比较后面的每一个元素都要和前面的元素比较，所以n个元素，需要和n-1个元素比较，移动的次数为0
比较的次数：n-1
移动的次数：0

2.最坏的情况（关键字在记录序列中逆序有序）：
1)当元素的初始序列为反序时，每趟排序中待插入的元素都要和[0,i-1]中的i个元素进行比较且要将这i个元素后移(arr[j+1] = arr[j])，i个元素后移移动次数当然也就为i了，再加上temp = arr[i]与arr[j+1] = temp的两次移动，每趟移动的次数为i+2,此时比较次数(Cmin)和移动次数(Mmin)达到最小值。 此时时间复杂度为O(n2)。

2）例子：
如：5 4 3 2 1
比较： 第2个元素和第1个元素比较
第3个元素和前2个元素比较
…
第n个元素和前n-1个比较
最后： 把1+2+…+n-1加起来求平均值
比较的次数：1+2+…+n-1 = (n+2)(n-1)/2
移动的次数：(n+4)(n-1)/2

3.事件复杂度结论：原始数据越接近有序，排序速度越快
1）最好的情况下（顺序有序）：O(n)
2）最坏的情况下（逆序有序）：O(n^2)
3）平均情况下，耗时差不多是最坏情况的一般：O(n^2)
4）要提高查找速度
减少元素的比较次数
减少元素的移动次数

  • 空间复杂度
    由直接插入排序算法可知，我们在排序过程中，需要一个临时变量存储要插入的值，所以空间复杂度为 O(1) 。

  • 算法稳定性
    直接插入排序的过程中，不需要改变相等数值元素的位置，所以它是稳定的算法。

  • 插入排序和选择排序的区别

1.插入排序和选择排序都有两层循环，外循环遍历整个数组
2.内循环稍有区别：
* 选择排序的内循环是遍历一组未排过序的数组。
* 插入排序的内循环是遍历一组已排过序的数组。

二分法插入排序

希尔排序

1.简介
希尔排序(Shell Sort)，一听这名字就知道是一个叫希尔的外国人发明的排序。没错，他就是唐纳德 希尔（Donald Shell），一位美国的计算机科学家，他于1959年发明的希尔排序算法。

2.基本思想
希尔排序是把序列按下标的一定增量分组，对每组使用直接插入排序算法排序；随着增量逐渐减少，每组包含的元素越来越多，当增量减至1时，再对整个序列进行一次直接插入排序。

3.希尔排序思路
1）定义增量序列D
2）对每个D进行“D-间隔”插入排序

4.希尔排序的特点
1）一次移动，移动位置比价打，跳跃式地接近排序后的最终位置
2）最后一次只需要少量移动
3）增量序列必须是递减的，最后一个必须是1
4）增量序列应该是互质的
