阿里云绑定个人域名

1. 注册域名
注册阿里云账号、并购买域名，具体流程可以自行百度，这里我就不多说了。

2. 解析域名
1）在终端ping一下之前配置的静态页面地址,可以获取到对应的ip地址:

2）@解析-Github
在cmd窗口中，ping之前配置的静态页面地址

记录类型选择：A
主机记录输入：@
解析线路：选择境外,因为我们是准备国内访问时访问到Coding，国外访问时访问到Github
记录值：输入刚才获取到的ip地址如:185.199.111.153
点击确定

3.www解析-Github

记录类型:选择CNME
主机记录输入:www
解析线路选择:境外
记录值就输入Github的静态页面地址:13718004742.github.io
点击确定。

3. 配置项目CNAME
1）在项目根目录-source目录下，新建CNAME文件,内容输入前面申请的域名,如：www.bboyzj.top
